🧪 MongoDB Chat Widget - Embedding Test

📋 Test Checklist

This page tests the embedding functionality of our MongoDB chat widget:

  • ✅ Widget loads from embed.js script
  • ✅ Chat bubble appears in bottom-right corner
  • ✅ Clicking bubble opens/closes chat window
  • ✅ Chat connects to API and responds to messages
  • ✅ Custom configuration works (title, colors, etc.)

🔧 Current Configuration

The widget on this page is configured with:

<script 
  src="http://localhost:3000/embed.js"
  data-mongodb-chat
  data-api-url="http://localhost:8000"
  data-title="Test Assistant"
  data-subtitle="Testing the embedding system"
  data-primary-color="#6f42c1"
  data-position="bottom-right">
</script>

✅ How to Test

  1. Start the Backend: Run python start_server.py
  2. Start React Dev Server: Run npm start in mongodb-chat-widget folder
  3. Open this page: The chat bubble should appear automatically
  4. Test the chat: Click the bubble and try asking MongoDB questions

🎯 Test Questions

Try these questions to test the chat functionality:

  • "What are MongoDB best practices?"
  • "How do I create an index?"
  • "What is aggregation in MongoDB?"
  • "Hello" (test non-MongoDB questions)

🚀 Integration Success!

If you can see the chat bubble and it responds to your messages, the embedding system is working perfectly!

You can now use this same script tag on any website to add the MongoDB chat widget.
