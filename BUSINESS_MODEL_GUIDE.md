## 📋 **Customer Integration Process**

### **For Customers (Super Simple):**

1. **Sign Up** → Get API key
2. **Copy Code** → One line of HTML
3. **Paste & Go** → Widget appears instantly

```html
<!-- Customer just adds this to their website -->
<script
  src="https://your-domain.com/widget.js"
  data-widget-key="customer-api-key"
></script>
```

### **Customization Options:**

```html
<script
  src="https://your-domain.com/widget.js"
  data-widget-key="customer-api-key"
  data-title="Custom Title"
  data-primary-color="#007bff"
  data-position="bottom-left"
  data-welcome-message="Custom welcome message"
></script>
```
