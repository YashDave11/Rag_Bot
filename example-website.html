🔧 Easy Integration

Adding this chat widget to your website is as simple as adding one script tag:

<script src="http://localhost:3000/embed.js"
  data-mongodb-chat
  data-api-url="http://localhost:8000"
  data-title="MongoDB Assistant">
</script>
